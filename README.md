# Bucket

# 앱이름: HomePT

## 📖 프로젝트 개요
스마트글래스를 활용하여 사용자의 운동 자세를 실시간으로 분석하고 피드백을 제공하는 AR 피트니스 보조 서비스입니다.
본 프로젝트는 사용자가 강의 영상과 자신의 운동 자세를 비교할 수 있도록 하며, 올바른 자세 정보를 제공하여 운동 자세 교정에 도움을 줍니다.
